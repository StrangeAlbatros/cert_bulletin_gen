## 1. Specification technique

### Support

- Linux
- MacOS X

### Prérequis

- latex
- python

#### Linux debian like

```
sudo apt install python3 texlive texlive-formats-extra texlive-lang-french
```
#### Linux red hat like
```
sudo yum install python3 texlive texlive-formats-extra texlive-lang-french
```
